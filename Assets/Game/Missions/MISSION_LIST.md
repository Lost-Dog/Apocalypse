## ðŸ—ºï¸ Required Location Triggers

To complete all missions, create trigger zones at these locations:

1. **Base** - Central hub (multiple missions)
2. **NorthCheckpoint** - Mission 10
3. **Mall** - Mission 13
4. **WeaponCache** - Mission 19
5. **Bunker** - Mission 25
6. **EvacuationPoint** - Mission 30 (finale)

**Setup:**
```csharp
// Add this script to empty GameObjects at each location
public class MissionLocationTrigger : MonoBehaviour
{
    public string locationName = "Base";
    
    void OnTriggerEnter(Collider other)
    {
        if (other.CompareTag("Player"))
        {
            MissionManager.Instance.OnLocationReached(locationName);
        }
    }
}
```
