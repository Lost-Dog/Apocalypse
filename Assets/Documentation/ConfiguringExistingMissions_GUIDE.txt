  CONFIGURING EXISTING MISSIONS WITH MISSION ZONES - COMPLETE GUIDE
═══════════════════════════════════════════════════════════════════

Your existing missions can now use the Universal Mission Setup system!

This guide shows you how to configure all your existing ChallengeData 
assets (Supply Drops, Civilian Rescues, Boss Encounters, etc.) with 
custom Mission Zones and spawn points.

═══════════════════════════════════════════════════════════════════
QUICK START (5 MINUTES)
═══════════════════════════════════════════════════════════════════

STEP 1: Open the Configurator
─────────────────────────────
Top Menu → Division Game → Challenge System → Configure Existing Missions


STEP 2: Set Default Prefabs
────────────────────────────
At the top of the window, drag your prefabs:
  • Default Enemy: Your rogue AI prefab
  • Default Civilian: Your friendly civilian prefab
  • Default Boss: Your boss enemy prefab
  • Default Loot Box: Your supply crate prefab


STEP 3: Select Challenges
──────────────────────────
You'll see a list of ALL your challenges:
  • World Events (14 challenges)
  • Daily (8 challenges)
  • Weekly (6 challenges)

Click checkboxes OR use quick buttons:
  • Select All Supply Drops
  • Select All Civilian Rescue
  • Select All Boss Fights


STEP 4: Create Mission Zones
─────────────────────────────
Click: Create Mission Zones for Selected (X)


STEP 5: Position & Customize
─────────────────────────────
1. Find zones in Hierarchy under MissionZones (they'll be grayed out)
2. Move zones to map locations (can edit while inactive)
3. Customize spawn points as needed

IMPORTANT: Zones are created INACTIVE and will be automatically
activated by ChallengeManager when challenges spawn!

DONE! Your challenges now use custom spawn points! ✓

═══════════════════════════════════════════════════════════════════
WORKFLOW EXAMPLES
═══════════════════════════════════════════════════════════════════

EXAMPLE 1: Configure All Supply Drops
──────────────────────────────────────
Goal: Set up all 3 Supply Drop missions with spawn points

1. Open configurator
2. Set Default Enemy = Rogue AI prefab
3. Set Default Loot Box = Supply Crate prefab
4. Click "Select All Supply Drops" button
5. Click "Create Mission Zones for Selected (3)"
6. Result:
   MissionZones/
   ├─ Small_Supply_Cache (Easy: 4 enemies, 1 loot)
   ├─ Supply_Drop_Medium (Medium: 6 enemies, 2 loot)
   └─ Supply_Drop_Hard (Hard: 8 enemies, 2 loot)


EXAMPLE 2: Configure All Boss Encounters
─────────────────────────────────────────
Goal: Set up all 3 Boss fights

1. Set Default Enemy = Elite Guard prefab
2. Set Default Boss = Commander Boss prefab
3. Set Default Loot Box = Epic Loot Crate
4. Click "Select All Boss Fights"
5. Create zones
6. Result:
   MissionZones/
   ├─ Rogue_Lieutenant (Boss + 6 guards)
   ├─ Elite_Commander (Boss + 8 guards)
   └─ War_Chief (Boss + 10 guards)


EXAMPLE 3: Configure Everything at Once
────────────────────────────────────────
Goal: Create zones for ALL challenges

1. Set all 4 default prefabs
2. Don't select anything
3. Click "Create All Missing Zones"
4. Confirm dialog
5. Result: All 28 challenges now have Mission Zones!


═══════════════════════════════════════════════════════════════════
POSITIONING YOUR MISSION ZONES
═══════════════════════════════════════════════════════════════════

After creating zones, position them in your scene:

METHOD 1: Spread Around Map
────────────────────────────
1. Open Hierarchy → MissionZones
2. Select first zone (e.g., Small_Supply_Cache)
3. Move to location on map (use Scene view)
4. Repeat for each zone


METHOD 2: Cluster by Type
──────────────────────────
Downtown Area:       Supply Drop missions
Residential Area:    Civilian Rescue missions
Industrial Zone:     Boss Encounter missions
Outskirts:          Extraction Defense missions


METHOD 3: Difficulty-Based Placement
─────────────────────────────────────
Safe Zones (Level 1-3):      All Easy difficulty
Mid-Tier Areas (Level 4-7):  All Medium difficulty
Dangerous Zones (Level 8+):  All Hard/Extreme


═══════════════════════════════════════════════════════════════════
CUSTOMIZING SPAWN POINTS
═══════════════════════════════════════════════════════════════════

After zones are created, customize individual spawn points:

1. Find the Zone
   Hierarchy → MissionZones → YourMissionName

2. Expand to See Spawn Points
   Small_Supply_Cache
   ├─ ZoneMarker (visual disc)
   └─ SpawnPoints
       ├─ Enemy_01
       ├─ Enemy_02
       ├─ Enemy_03
       ├─ Enemy_04
       └─ LootBox_01

3. Customize Position
   • Move spawn points to custom locations
   • Rotate to set facing direction
   • Add/Remove spawn points manually

4. Change Prefabs
   Select MissionZone → Inspector → Spawn Points:
   Enemy_01
   ├─ Category: Enemy
   └─ Prefab Override: [Change to Sniper AI]

5. Advanced Settings
   Enable "Use Custom Settings" on any spawn point:
   ├─ Require NavMesh: ✓
   ├─ Random Rotation: □
   ├─ Fixed Rotation: (0, 90, 0)
   └─ Priority: 10


═══════════════════════════════════════════════════════════════════
INTEGRATION WITH CHALLENGEMANAGER
═══════════════════════════════════════════════════════════════════

Your configured zones automatically work with the challenge system:

AUTOMATIC FLOW
──────────────
1. ChallengeManager decides to spawn "Small Supply Cache"
   ↓
2. ChallengeSpawner receives spawn request
   ↓
3. Spawner finds nearest MissionZone of type "SupplyDrop"
   ↓
4. Spawner uses zone's custom spawn points
   ↓
5. Enemies and loot spawn at your positions!


FALLBACK BEHAVIOR
─────────────────
If no MissionZone is found near the spawn location:
  • System uses flexible spawning from ChallengeData
  • Spawns randomly around challenge position
  • Still works, just not with custom placement


═══════════════════════════════════════════════════════════════════
TIPS & BEST PRACTICES
═══════════════════════════════════════════════════════════════════

DO:
✓ Set default prefabs before creating zones
✓ Position zones in logical map locations
✓ Test each mission type to ensure spawns work
✓ Use visual markers to see zone boundaries
✓ Group similar mission types in same area
✓ Adjust spawn points after testing gameplay

DON'T:
✗ Create zones without setting prefabs first
✗ Leave all zones at world origin
✗ Overlap mission zones of different types
✗ Delete SpawnPoints folder (breaks the zone)
✗ Forget to save scene after positioning

═══════════════════════════════════════════════════════════════════
TROUBLESHOOTING
═══════════════════════════════════════════════════════════════════

Problem: "All selected challenges already have zones"
Solution: Challenges already have Mission Zones. Either:
          • Select different challenges
          • Delete existing zones to recreate them

Problem: Zones created but can't find them
Solution: Look in Hierarchy → MissionZones

Problem: Spawn points not showing in scene
Solution: • Enable Gizmos in Scene view (top right)
          • Select the MissionZone to see highlights

Problem: Enemies not spawning at custom points
Solution: • Check zone is near challenge spawn location
          • Ensure mission types match
          • Verify prefabs assigned to spawn points

Problem: Zone has wrong number of spawn points
Solution: • Manually add/remove spawn points
          • Or delete zone and recreate
